# STEEL FRONT — Mobile 3D FPS

Three.js 製のシングルファイル 3D FPS ゲーム(バトルフィールド風)。
PC・スマートフォンの両方に対応しています。

## 遊び方

`index.html` をブラウザで開くだけでプレイできます(ビルド不要)。

### PC 操作
| 操作 | キー |
|---|---|
| 移動 | W / A / S / D |
| 視点 | マウス(画面クリックでポインターロック) |
| 射撃 | 左クリック |
| ダッシュ | Shift(前進中) |
| ジャンプ | Space |
| リロード | R |
| 乗り物に乗る / 降りる | E |

### スマートフォン操作
- **左画面ドラッグ**: 移動(前いっぱいに倒すとダッシュ)
- **右画面ドラッグ**: 視点操作
- **FIRE / JUMP / RELOAD** ボタン
- 乗り物に近づくと「**乗る**」ボタンが出現(ジープ・戦車に搭乗可能)

## 主な機能
- ウェーブ制のサバイバル戦闘(敵はウェーブごとに増加)
- 乗り物: M151 ジープ(重機関銃)/ T-70 戦車(主砲)
- レーダー・キルフィード・ヒットマーカーなどの HUD
- WebAudio による効果音(外部アセット不要)
- WebGL コンテキストロスト自動復旧 / エフェクト完全プール化による負荷対策

## バージョン履歴

### v0.1.1(2026-07-11)
バグ修正・改善:
- **移動方向の修正**: 横を向いた状態で移動すると前後(上下入力)が反転して逆方向に移動してしまうバグを修正
  - カメラ空間 → ワールド座標への回転変換の符号誤りを修正(`updatePlayer` 内の `wx` / `wz` 計算)
- **戦車の砲塔の向きを修正**: 砲塔がエイム方向と 180° 逆を向いていた問題を修正(余分な `+Math.PI` を除去)
  - ジープの機銃塔も同じ計算を共有しているため、あわせて正しい向きになります
- **AI(CPU)の動きを改善**:
  - 障害物回避: 進行方向が塞がれている場合、左右にスライドして壁沿いに回り込むように
  - スタック検知: 一定時間ほぼ移動できていない敵は自動的に回避行動へ移行
  - 視線(LoS)に基づく行動: プレイヤーが見えない間は距離を保たず接近・回り込みを行う
  - エイム時間の導入: 視線が通った直後に即撃ちせず、約 0.35 秒のエイム時間を挟むことで理不尽さを緩和
  - 視線判定を間引き(約 0.25 秒間隔)して処理負荷を軽減
