# STEEL FRONT — 更新履歴

## v0.3.1(2026-07-11)
- 狭い隙間をAIが通れない問題を修正(当たり半径縮小・左右先読み回避・壁際往復の防止)
- AIもジャンプ可能に(低い障害物を飛び越え、段差の上にも乗れる)
- ヘリコプターが飛行中に壊れると浮いたままになる問題を修正(きりもみ墜落→着地で大爆発)
- ヘリコプターの視点を改善(ピッチ可動域拡大・機体を画面下寄りに配置して前方視界を確保)
- パラシュートを追加(高高度からの落下で自動展開/ヘリから空中脱出可能/落下ダメージ導入)
- マップの高低を修正(細かい起伏のガタつきを低減)
- 道路を配置(センターライン付きの実体路面メッシュ+道路に沿った地形の平滑化+北側幹線を追加)
