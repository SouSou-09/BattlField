# STEEL FRONT — 更新履歴

## v0.5.4(2026-07-12)
- 車両HPとエンジン損傷に応じて軽微な灰煙、濃い黒煙、発火へ段階的に悪化する損傷表現を追加
- 戦車の左右履帯を個別部位化し、破損後は前後移動不能・その場旋回のみ可能な状態と修理を実装
- ヘリのローター重大損傷によるオートローテーション不能、ボートのエンジン破損による推進力喪失と慣性漂流を追加

## v0.5.3(2026-07-12)
- AIが近くの空車両を確保し、分隊員を砲手席・同乗席へ乗せて拠点移動と戦闘へ活用する判断を追加
- AI運転手が目標拠点または近距離の敵へ自律走行し、障害物・深水を避けながら車載砲手と連携
- 足元へ着弾した敵グレネードを検知し、安全な方向への緊急離脱、逃げ場がない場合の伏せ反応を追加

## v0.5.2(2026-07-12)
- 正面戦闘が4秒以上膠着したAIが左右の安全な側面へ回り込むフランキング判断を追加
- AI分隊へ突撃・偵察・援護・狙撃の役割を割り当て、偵察役の先行と援護役の後方支援隊列を実装
- AIにも弾倉・予備弾・リロードを導入。完全な弾切れ時は距離に応じて撤退または近接攻撃へ切り替え

## v0.5.1(2026-07-12)
- Jキーの携行RPGを追加。正面・側面・後部装甲で貫通倍率が変化し、背面攻撃ほど車両へ有効
- Yキーで設置、Uキーで手動起爆できるクレイモアを追加。設置後は前方近接センサーでも自動起爆
- 各陣営のAI狙撃兵が高所の固定陣地へ移動し、伏せ姿勢で射程・命中精度を高めて粘る行動を追加

## v0.5.0(2026-07-12)
- 偵察兵のQ短押しタグを追加。視認した敵を一定時間マーキングし、分隊共有HUDとマップへ反映
- Kキーの双眼鏡を追加。8倍相当の望遠視界からQタグと連動して最大360mの索敵支援が可能
- 通常歩行・疾走・しゃがみ・伏せごとの足音到達範囲をミニマップへパルスリングと距離で表示

## v0.4.8(2026-07-12)
- ジープ/ヘリ/ボートのフロントガラスに部位HPと段階的なひび割れ・破砕表現を追加。小火器や爆風の命中部位が見た目へ反映
- ジープ/バイクのタイヤを個別に破壊可能化。パンクした側へ車体が傾き、片輪走行時は最高速と旋回性能が低下。リペアツールで部位修理可能
- 進入可能建物・二階建て・廃墟の壁セグメントを爆破可能化。戦車砲やヘリロケット等でMesh/AABBを除去し、実際に通れる侵入口を作成可能

## v0.4.7(2026-07-12)
- プレイヤー用スモークグレネードを追加(Nキー / SMKボタン、出撃ごとに2個)。煙幕がhasLineOfSightを直接遮断し、拠点突入や蘇生を支援
- AIのボイス/無線を追加。敵発見、拠点防衛、前進・援護指示を状況に応じたテキスト無線と通信音で通知
- AIの制圧射撃と至近弾サプレッションを追加。遮蔽への制圧射撃、プレイヤーの照準ブレ、AIの怯み/命中低下を実装(援護兵は耐性あり)

## v0.4.6(2026-07-12)
- Q/E長押しのリーンを追加。視点と射線を左右へ移動して壁際から最小露出で射撃可能
- Spaceで窓枠・低い壁を判定し、前方へ乗り越えるボルトアクションを追加
- HP0で16秒のダウン状態へ移行し、近くの分隊AIまたはF長押し応急処置で蘇生可能。失血時のみチケット消費
- M249 LMG(100発・高反動・伏せ撃ち安定)を追加し、援護兵の標準装備に設定

## v0.4.5(2026-07-12)
- 物理薬莢プールを追加し、発砲時に排莢して地面で跳ねながら転がる表現を実装。マズルライトも強化
- 一人称の両腕モデルとリロード/疾走に連動する武器モーションを追加
- 死亡時に射撃位置・距離を3秒表示するキルカメラを追加
- 低HP時の血の脈動、心音、視点揺れ、足を引きずる移動速度低下を追加
- リザルトにMVPハイライト、最多キル武器、兵科、最高連続キル、拠点貢献度を演出表示

## v0.4.3(2026-07-12)
- 突撃兵/衛生兵/援護兵/偵察兵の兵科システムを追加。武器と専用ガジェットをセット運用(Bキー)
- 味方AI分隊へ自動参加し、安全な分隊員の現在位置をデプロイ画面の出撃地点として選択可能
- 連続3キルでUAV、5キルで迫撃砲支援、7キルで補給物資投下のスコア報酬を追加

## v0.4.2(2026-07-12)

- B/C/D/地下壕を結ぶ地下トンネル網を追加。入口はミニマップ非表示でEキーから奇襲移動
- 草むら・茂みを追加。伏せ状態で入ると敵AIから大幅に発見・命中されにくい隠密状態

- 弾痕・爆発クレーター・焦げ跡を最大140個まで戦場に蓄積し、リスタート時にリセット

## v0.4.1(2026-07-12)

- バイク追加(KLR バイク — 高速26m/s・無防備・2人乗り。後席は自分の武器で射撃可能。両HQ+拠点B/Cに配置で旗の裏取りが可能に)

## v0.4.0(2026-07-12)
